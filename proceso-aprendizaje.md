# Proceso de aprendizaje previo

Antes de desarrollar la práctica final *Terraformando JavaScriptia*, el proyecto se apoya en un recorrido progresivo de aprendizaje en programación JavaScript, estructurado en tres bloques principales (PEC 1, 2 y 3) dentro del Grado en Multimedia (UOC).

Estas prácticas no se incluyen como código en este repositorio porque su foco era principalmente teórico y de entrenamiento de fundamentos, pero son clave para entender el planteamiento y la resolución del proyecto final.

---

## PEC 1 · Tipos de datos y estructuras algorítmicas

**Contenidos trabajados:**
- Pensamiento algorítmico y buenas prácticas
- Tipos de datos básicos
- Operadores y prioridades
- Condicionales (`if / else`)
- Bucles (`while`, `for`)
- Primeros ejercicios en JavaScript

**Objetivo:**  
Interiorizar la lógica básica de programación y la traducción de algoritmos a JavaScript.

---

## PEC 2 · Funciones y arrays

**Contenidos trabajados:**
- Definición y uso de funciones
- Parámetros y valores de retorno
- Reutilización de código
- Arrays unidimensionales y bidimensionales
- Recorridos con bucles
- Cálculos agregados (sumas, máximos, mínimos, medias)

**Objetivo:**  
Aprender a estructurar código reutilizable y trabajar con colecciones de datos.

---

## PEC 3 · Objetos y esquemas de recorrido

**Contenidos trabajados:**
- Objetos JavaScript (propiedades)
- Arrays de objetos
- Acceso a datos complejos
- Recorridos y búsquedas
- Manipulación de strings sin métodos avanzados

**Objetivo:**  
Trabajar con estructuras de datos similares a las usadas en aplicaciones reales, preparando el terreno para el uso de objetos complejos como `planetData`.

---

## Relación con la práctica final

La práctica final integra todos estos conceptos en un único proyecto:

- `planetData` → objeto complejo con múltiples niveles
- Funciones puras → lógica desacoplada de la interfaz
- Arrays y recorridos → análisis y transformación de datos
- Condicionales → activación de sistemas del planeta

Este repositorio muestra el resultado de ese proceso aplicado a un proyecto completo.
